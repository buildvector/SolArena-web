# SolArena Tokenomics

SolArena is built around sustainable on-chain game activity.
The token is designed to support competition, burn mechanics, and long-term engagement.

Core mechanics:
- PvP-based fee generation
- Burn-to-boost multiplier
- Leaderboard scoring model
- No promised airdrops
- No revenue guarantees

Token Purpose:
- Used for ecosystem alignment
- Used for burn-based multiplier
- Used as competitive signal
- Used for leaderboard amplification

There are no:
- Guaranteed yields
- Staking promises
- Fixed ROI mechanics
- Artificial emissions

---

## PvP Fee Model

- Each PvP game generates a small fee
- Fees are transparent and predictable
- Activity drives ecosystem visibility
- Volume comes from gameplay, not inflation

The platform is designed so:
More games = more activity  
More activity = more burn incentives  
More burn = higher leaderboard competition  

---

## Burn Multiplier

Players may burn tokens voluntarily.

Burning tokens:
- Permanently reduces total supply
- Increases player multiplier
- Boosts leaderboard scoring

Early burns give stronger boost.
The multiplier curve follows a diminishing returns model.

Example concept:
- First 1,000 tokens burned → strong multiplier increase
- 10,000 tokens burned → smaller relative increase
- 100,000 tokens burned → marginal improvement

This prevents:
- Immediate whale domination
- Infinite scaling advantage
- Pay-to-win imbalance

Burning improves influence — not guaranteed profit.

---

## Leaderboard Scoring Model

Leaderboard score is calculated from:

- PvP Wins
- Fee Contribution
- Win Streak Bonus
- ELO Skill Rating
- Burn Multiplier

Final Score =
(Base Points + Skill Modifier + Fee Points) × Burn Multiplier

Skill remains primary.
Capital enhances — it does not replace skill.

---

## Supply Philosophy

- Fixed initial supply
- No hidden mint functions
- No backdoor inflation
- Public mint address
- Fully transparent burn mechanics

Supply can only decrease through burn.

---

## Economic Design Philosophy

SolArena is not designed as:
- A pump-and-dump token
- A yield farm
- A short-term extraction scheme

It is designed as:
- A competitive PvP economy
- A season-based engagement engine
- A burn-driven scarcity model

Long-term value, if any, must come from:
- Usage
- Community
- Competition
- Transparency

Nothing else.
